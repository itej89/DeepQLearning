# Deep Q-Learning

This project contains full implementation of Deep Q-Learning for Open AI gym environments


## Usage

```sh
git clone https://github.com/itej89/DeepQLearning.git
```

### Traing the model
```bash
$ python RL.py
```

### Testing the model
```bash
$ python TestModel.py
```
